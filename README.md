## Descripción General
Este repositorio centraliza el desarrollo de soluciones algorítmicas, simulaciones físicas y motores de juego implementados en **C++**. Cada proyecto refleja el avance en la adquisición de competencias en gestión de memoria, estructuras de datos, programación orientada a objetos (POO), y el uso eficiente de la librería estándar (STL) y frameworks industriales como Qt6.
